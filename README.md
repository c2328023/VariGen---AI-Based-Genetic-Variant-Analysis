# VariGen: AI-Based Genetic Variant Classification

This project is developed for the TEKNOFEST 2026 Health in AI Competition (University Level). The objective is to classify genetic variants as Pathogenic or Benign using machine learning techniques.

## Project Scope
The project focuses on predicting the clinical significance of genetic mutations. Due to competition rules, genomic coordinates are hidden to ensure the model focuses on biological and bioinformatics features rather than data memorization.

## Technical Details
The classification model utilizes a comprehensive feature set including:
- Amino acid changes and sequence context.
- Biochemical property alterations.
- Evolutionary conservation scores.
- Population frequency data.

The performance of the model is evaluated based on the F1-Score to ensure a balance between precision and recall in clinical predictions.
